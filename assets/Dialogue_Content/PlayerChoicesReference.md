# Player Choice Reference

This file lists the player-facing dialogue choices used in the plot.

## Final reveal choices
- What is this? What happened to me?
- A simulation? This is a simulation?
- I volunteered? I don’t remember that.
- I want to go back to my body.
- I want the clearest, most efficient outcome.

## Subquest choice patterns
- Toy: comfort vs criticism
- Rabbit: heartfelt letter vs strict formal letter
- Old Man: preserve memory vs fix for accuracy
