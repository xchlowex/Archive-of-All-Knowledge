# Archive-of-All-Knowledge
This is a 2D pixel RPG created for the course COMP3329.

### Game Description
You, as the player, wake up on a deserted central island. The only thing you see is a board — “Archive of All Knowledge” with dim stars, and the only goal in your mind: light them up. There are other mysterious islands, each with a quest you have to complete. Besides these quests, there seems to be some other NPCs who are waiting for your help. Would you try to build connections to grasp even the smallest idea of what this world is about, or rush through your journey for your own escape? What awaits you after the stars are all lit up?
