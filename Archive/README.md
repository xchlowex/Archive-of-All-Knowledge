# Archive Branch: Previous Unity Test Project

This branch stores Chloe's previous Unity test project as a stable archive.

It includes workable scenes for:
- 1 Central Island scene
- 2 NLP Island scenes
- 3 Vision Island scenes

This branch also contains all related assets and game objects required by those scenes.

Archive scope: this branch contains only the `Assets`, `Packages`, and `ProjectSettings` folders.

## Purpose

The goal of this branch is to preserve an earlier, functional version of the project for reference, testing comparison, and recovery if needed.
